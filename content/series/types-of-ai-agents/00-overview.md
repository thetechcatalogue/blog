## What Is an AI Agent

An AI agent is a system that perceives its environment, reasons about what to do, and takes action. The difference between agent types lies in how much they can perceive, remember, plan, and collaborate.

## Six Architectures

We cover six types in this series. Reflex agents map inputs to actions through fixed rules. Tool-use agents run a reasoning loop that calls external tools. Planning agents decompose a goal into steps before executing anything. Memory-augmented agents retrieve context from short and long-term stores. Multi-agent systems delegate to specialized agents running in parallel. And autonomous agents run a self-directed loop with no human checkpoints.

## The Complexity Spectrum

These six types sit on a spectrum from simple to capable. Simpler agents are predictable and cheap. More capable agents handle ambiguity and long-horizon tasks but require more careful design around reliability and safety.
