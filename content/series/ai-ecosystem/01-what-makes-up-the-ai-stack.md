## The AI Stack Is Not Just the Model

A production AI system is a stack. The model is one layer, but everything around it determines whether the experience is useful, safe, fast, and maintainable.

## Core Layers

Most AI products combine a model layer, prompting layer, context layer, orchestration layer, evaluation layer, and an application layer that users actually interact with.

## Models

Models generate text, code, images, or structured outputs. Choosing a model is a tradeoff across quality, latency, context window, cost, and controllability.

## Context and Retrieval

Raw model knowledge is rarely enough. Retrieval adds fresh or private data so the model can answer with more relevant context.

## Tools and Agents

Tools let the model act: search, call APIs, query data, or update systems. Agents add planning and decision-making around those tool calls.

## Evaluation

Without evaluation, teams end up shipping based on intuition. Good AI systems measure correctness, relevance, latency, and failure patterns continuously.

## Product Experience

The visible product still matters. A strong AI system needs clear affordances, fallback paths, and interaction design that helps users trust what they see.

## Summary

The AI ecosystem is a layered system. The model is central, but the real product emerges from how context, orchestration, evaluation, and UX fit together.
